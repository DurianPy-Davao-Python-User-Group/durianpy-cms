# API Overview

The DurianPy CMS provides two powerful ways to interact with your data: a **REST API** and a **GraphQL API**.

### 🔗 Base URLs
*   **Local Development**: `http://localhost:3000/api`
*   **Production**: `https://cms.durianpy.org/api`

### 🛠️ API Types
1.  **REST API**: Available at the base URL (e.g., `/api/posts`).
2.  **GraphQL API**: Available at `/api/graphql`. Includes a built-in Playground for testing queries in the browser.

For deep technical details, refer to the official guides:
