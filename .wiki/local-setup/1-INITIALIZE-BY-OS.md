# Prerequisites by OS

This project relies on Docker and DevContainers. Install the requirements for your operating system:

### Windows

1. Install [WSL 2](https://learn.microsoft.com/en-us/windows/wsl/install).
2. Install [Docker Desktop](https://www.docker.com/products/docker-desktop/) and ensure WSL 2 integration is enabled in settings.

### macOS

1. Install [Docker Desktop](https://www.docker.com/products/docker-desktop/)

### Linux

1. Install [Docker Desktop](https://www.docker.com/products/docker-desktop/) or install Docker Engine and Docker Compose using your distribution's package manager.
   - _Example (Ubuntu):_ `sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin`
